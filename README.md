# Sidebar Menu (HTML & CSS)

## 📌 Overview
A simple sidebar navigation menu built using HTML and CSS only.  
Uses the `:checked` pseudo-class to toggle the menu without JavaScript.

## ✨ Features
- Pure HTML & CSS  
- Sidebar toggle (no JS)  
- Smooth transitions  
- Font Awesome icons  

## 📁 Files
```
index.html
style.css
camera.jpg
```

## ▶️ Run
Open `index.html` in your browser.
